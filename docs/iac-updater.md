# IaC Updater

To automate the update of Infrastructure of Code (IaC) a specific Github App has been created.

This Github App has the following characteristics:

- **Github App Name**: `IaC Updater`
- **Description**: `Automate creation of Pull Requests to update Infrastructure of Code (IaC) configurations.`
- **Homepage URL**: `https://github.com/bgaillard`
- **Webhook**: `Disabled`
- **Repository permissions**:
    - **Contents**: `Read & Write`
    - **Pull requests**: `Read & Write`
- **Where can this GitHub App be installed?**: `Only on this account`
